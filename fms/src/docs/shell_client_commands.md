# Commands

### pin
#### set &lt;pin&gt;
"Success" or "Failure" based on whether the pin was set successfully.
#### get
Returns a boolean indicating whether the pin is set.

### turn
#### on &lt;pin&gt;
"FMS turning on" or "Invalid pin" based on whether the pin is valid and the FMS is turning on.
#### off &lt;pin&gt;
"FMS turning off" or "Invalid pin" based on whether the pin is valid and the FMS is turning off.
